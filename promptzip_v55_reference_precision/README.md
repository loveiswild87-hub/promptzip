# PromptCraft V55 Reference Precision

V55는 사용자가 제시한 참고 메인 화면에 더 가깝게 히어로 섹션을 정리한 버전입니다.

## 핵심 변경
- 참고 이미지에 맞춰 좌측 카피 / 우측 데모 카드 구도 재정렬
- 헤드라인 마지막 단어가 가려지지 않도록 텍스트 안전 영역 확보
- 입력 카드, 브릿지, 결과 카드의 수평 축을 더 안정적으로 보정
- 오브, 스파클, 회전 링, 빛 흐름, 브릿지 펄스, 플랫폼 발광 등 모션 요소 강화
- 모바일/태블릿 구간에서도 겹침이 줄어들도록 반응형 재조정
