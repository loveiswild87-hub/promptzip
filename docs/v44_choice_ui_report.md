# PromptZip V44 선택 안내 UI 보정 보고서

## 목적
고객 화면에서 `PromptZip`과 `PromptCraft`가 문장 안에 배지 형태로 섞여 보이던 문제를 개선했다. 기존 구조는 문장 흐름과 배지가 충돌해 가독성이 낮았기 때문에, 두 기능을 선택 카드로 분리했다.

## 수정 내용
- 4단계 흐름의 1번 카드에서 문장형 안내를 제거하고 카드형 비교 UI 적용
- `PromptZip`은 파란 Zip 카드, `PromptCraft`는 민트 Craft 카드로 분리
- 무료 체험 영역에도 동일한 선택 안내 카드 적용
- FAQ의 기능 차이 설명도 문장형에서 카드형 비교로 수정
- 기존 기능 및 과금 옵션은 추가/삭제 없이 유지

## 기대 효과
- 처음 방문한 고객이 두 기능의 차이를 더 빠르게 이해
- 서비스명 배지가 문장 흐름을 끊는 문제 감소
- 선택 흐름이 “읽는 설명”에서 “고르는 안내”로 전환되어 사용성 개선
