# PromptZip V37 디자인 정리 보고서

## 목적

V37은 V36의 기능을 유지하면서 사용자가 처음 들어왔을 때 실제 유료 서비스처럼 느낄 수 있도록 시각적 설득력을 높인 버전입니다.

## 개선 사항

1. 히어로 영역의 메시지와 CTA 강화
2. 절감률 미리보기 카드의 정보 밀도 개선
3. PromptZip / PromptCraft / Engine Cost Options 구분 카드 추가
4. 입력창, 결과창, 요금제 카드의 여백과 글자 배치 정리
5. 설명 문단과 예시 문단에 양쪽 정렬 적용
6. 고급 과금 옵션은 접기/펼치기 구조 유지
7. 모바일 화면에서 카드가 한 줄 배치로 자연스럽게 전환되도록 유지

## 유지 기능

- OpenAI / Claude / Gemini 선택
- 캐시 적용 비율
- Thinking/Reasoning 추가 출력
- 예상 출력 배율
- 검색/그라운딩 호출 수
- 도구 호출 수
- Batch 할인
- 지역 프리미엄
- PromptZip / PromptCraft 분리 상품 구조
