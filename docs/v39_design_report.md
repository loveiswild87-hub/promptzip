# V39 Design Report

## 목표

고객이 처음 접속했을 때 기능이 복잡해 보이지 않도록, 사용 흐름과 정보 배치를 정리했습니다.

## 수정 내용

1. 상단 히어로 문구를 “AI 질문은 더 정교하게, AI 비용은 더 가볍게”로 정리했습니다.
2. 사용 흐름을 4단계 카드로 분리했습니다.
3. PromptZip과 PromptCraft의 차이를 고객 관점에서 설명했습니다.
4. 입력 영역 옆에 결과 확인 기준을 명확히 배치했습니다.
5. 고급 과금 옵션은 접기/펼치기 구조로 유지했습니다.
6. FAQ와 설명 문단에 양쪽 정렬을 적용했습니다.

## 유지한 것

- 기존 압축 API
- PromptCraft API
- OpenAI / Claude / Gemini 엔진 선택
- 캐시, thinking/reasoning, 검색/도구 호출, batch 할인, 지역 프리미엄 옵션
- 관심 등록 CSV 저장
